# Quickstart

To start using GQty, run `@gqty/cli` with your favourite packager.

<Alert className="mt-5 mb-3" severity="success" title="Even Quicker Start!">
  Run the following command with your endpoint as an argument to skip the
  interactive setup and jump right in!
</Alert>

<Tabs>
  <View paddingBottom={3}>
    <Tabs.List>
      <Tabs.Item value="1">npm</Tabs.Item>
      <Tabs.Item value="2">yarn</Tabs.Item>
      <Tabs.Item value="3">pnpm</Tabs.Item>
    </Tabs.List>
  </View>

  <Tabs.Panel value="1">

    ```bash copy
    > npx @gqty/cli
    ```

  </Tabs.Panel>
  <Tabs.Panel value="2">

    ```bash copy
    > yarn dlx @gqty/cli
    ```

  </Tabs.Panel>
  <Tabs.Panel value="3">

    ```bash copy
    > pnpm dlx @gqty/cli
    ```

  </Tabs.Panel>
</Tabs>

After the interactive setup, you should now see these files in your project:

```
.
├─ gqty
│ ├─ index.ts
│ └─ schema.generated.ts
└─ gqty.config.ts

```

You are all set! Head over to one of the following section to create your first
fetching component!
