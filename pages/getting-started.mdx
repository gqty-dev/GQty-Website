# Quickstart

To start using GQty, run `@gqty/cli` with your favourite packager.

<Alert className="mt-5" severity="success" title="Even Quicker Start!">
  Use your API endpoint as a CLI argument to skip the interactive setup and jump
  right in!
</Alert>

<Tabs items={["npm", "yarn", "pnpm"]}>
  <Tab>

    ```bash copy
    > npx @gqty/cli
    ```

  </Tab>
  <Tab>

    ```bash copy
    > yarn dlx @gqty/cli
    ```

  </Tab>

  <Tab>

    ```bash copy
    > pnpm dlx @gqty/cli
    ```

  </Tab>
</Tabs>

After the interactive setup, you should now see these files in your project:

```
.
├─ gqty
│  ├─ index.ts
│  └─ schema.generated.ts
└─ gqty.config.ts
```

You are all set! Head over to [React](/guides/react/read) or the [core](/guides/core/resolve)
to create your first fetching component!

</Alert>
